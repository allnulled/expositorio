Línea de comandos:
  • Alda-Autoplay

    Alda es un lenguaje de programación para hacer música. Con esta herramienta puedes escuchar tus programas con tan solo guardar el fichero.

  • Mongrator

    Migraciones para MongoDB.

  • MySQL2Laravel

    Convierte una conexión de base de datos MySQL a un proyecto Laravel mediante scripts en PHP.

  • Matchalize

    Encuentra textos y expresiones regulares de manera recursiva en ficheros, y ordena y extrae analíticas en formatos JSON en función del nombre o de la frecuencia de aparición.

  • Progene

    Herramienta para generar fácilmente generadores de proyecto 100% personalizables, configurables y rápidos de desarrollar y usar.

  • Muchos de los proyectos que se mencionan fuera de este grupo también tienen su herramienta de línea de comandos.
